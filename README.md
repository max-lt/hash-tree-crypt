Hash Tree Crypt
===========================

Hash tree crypt is an seekable stream cipher using a perfect binary tree to provide an infinite (quite large) one-time pad.

## Usage

### Build
```
cargo build --release
```

### Clean
```
cargo clean
```

### Run
```
cargo run --release -- -i input-file -o output-file
```

### Install
```
cargo install --path .
```
